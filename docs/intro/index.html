CoVid-19 疫情概览

病毒简介

病毒：SARS-CoV-2，其导致疾病命名 COVID-19

传染源：新冠肺炎的患者。无症状感染者也可能成为传染源。

传播途径：经呼吸道飞沫、接触传播是主要的传播途径。气溶胶传播和消化道等传播途径尚待明确。

易感人群：人群普遍易感。老年人及有基础疾病者感染后病情较重，儿童及婴幼儿也有发病

潜伏期：一般为 3～7 天，最长不超过 14 天，潜伏期内可能存在传染性，其中无症状病例传染性非常罕见

宿主：野生动物，可能为中华菊头蝠
